
Background Remover Tool

This is a simple web tool to remove the background from images using the Remove.bg API.

Steps to use:
1. Get your free API key from https://www.remove.bg/api
2. Open 'index.html' in a code editor.
3. Replace 'YOUR_REMOVE_BG_API_KEY' in the script section with your actual API key.
4. Upload the 'index.html' file to your web server.
5. Open the website, upload an image, and click "Remove Background" to get the processed image.

Note:
- Make sure you have internet connectivity as the tool calls the Remove.bg API.
- Free API has some usage limits.
